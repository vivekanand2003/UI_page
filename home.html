HTML Tags

div tag

The div tag is used as a container for HTML elements and defines a division or a section in an HTML document.

br & b tag

Use the br tag tag to enter line breaks, not to add space between paragraphs.
The b tag specifies bold text without any extra importance.

link,hr & a tag

The link tag defines the relationship between the current document and an external resource.
The hr tag represents a thematic break between paragraph-level elements.
The a tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the element is the href attribute, which indicates the link's destination.

nav, tag

The nav tag defines a set of navigation links. Notice that NOT all links of a document should be inside a nav element. The nav element is intended only for major blocks of navigation links.
The background-image property in Cascading Style Sheets (CSS) allows users to specify a background image for an element.to set a background image for a document, the user can type backgroundImage = "url('img_tree.png')"

flexbox, position

The Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex (expand) to fill additional space or shrink to fit into smaller spaces.
The possition css property sets how an element is positioned in a document.
